# COVID-19ワクチン接種状況

政府CIOポータルのオープンデータを基に、都道府県別のCOVID-19ワクチン接種状況を確認できるアプリケーションです。

## デモ
https://code4fukui.github.io/covid19vaccine/

## 機能
- 都道府県別のワクチン接種完了率と初回接種率の確認
- 全国平均のワクチン接種状況の確認
- データの毎日自動更新

## データ・API
- 新型コロナワクチンの接種状況（高齢者等） | 政府CIOポータル
  - https://cio.go.jp/c19vaccine_dashboard
- CSV形式のオープンデータ
  - 全件: https://code4fukui.github.io/covid19vaccine/latest.csv
  - 都道府県別サマリー: https://code4fukui.github.io/covid19vaccine/latest_summary.csv
